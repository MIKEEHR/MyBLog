WHAT IS A NETWORK TOPOLOGY ?

Topology is the way we arrange the network it includes the physical and logical description of how the nodes i.e computers are linked/related to each other.

STAR TOPOLOGY

This topology uses a hub which is a networking device used to facilitate communication like when we use a sharing app on our mobile devices or when we share screen on TV via a smart phone.
ADVANTAGES:
1> As each node is connected to the hub even if many nodes go down the server will remain and other nodes can facilitate.
2> Simple design.
DISADVANTAGES:
1> If the hub fails the whole system will fail due to single point error.
2> There is a limit to the number of nodes connected.

MESH TOPOLOGY

In a mesh network topology we have a point to point connection where all nodes are interconnected. nC2 links are needed in this topology where n is the no.of nodes
ADVANTAGES:
1>Single point failures do not affect the entire network.
2>Reliable and stable.
DISADVANTAGES:
1>Hard to configure.
2>Cost of cables is needed in mesh topology.

BUS TOPOLOGY

A physical connector called a Tap lies in the cable and connects the computer to it via it's backbone hence it is also called Line topology.
ADVANTAGES:
1>We can increase backbone length to connect more nodes.
2>More Scalable.
DISADVANTAGES:
1>If some nodes fail to transmit data errors in other nodes will be occured as there is only one cable.
2>Less reliable.

RING TOPOLOGY

In this network topology a data token is revolved in a circle, when one node sends the data it can be received by other nodes and that single node sends the data not the token, where each node has exactly two neighbours.
ADVANTAGES:
1>It is very scalable, easy to add new nodes.
2>Fault Isolation i.e easy to find faults in network.
DISADVANTAGES:
1>If one node goes down the whole system fails due to single point failure.
2>If more nodes are connected then there would be a delay in the network.

HYBRID TOPOLOGY

hyrbrid topologies combine two or more different topologies together and is a good example for integrating bus and star layouts. These structures in used in big companies.
ADVANTAGES:
1>Nice Flexibility .

DISADVANTAGES:
1>As each topology has it's disadvantages it is difficult to design a good hybrid topology.
2>Time and money are a must to design a good design.
